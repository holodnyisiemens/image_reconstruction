# image_reconstruction

## Восстановление исходного серого изображения с помощью восстановления распределений в координатах, соответствующих пикселям.

Генерируется N агентов и помещаются в случайные координаты нулевой матрицы, по размерам аналогичной исходному изображению.
За каждое посещение агентом какой-либо координаты, значение в ней увеличивается на 1.
Перед дальнейшим шагом каждого агента рассчитывается распределение в каждой соседней координате.
Агент перемещается в ту координату, в которой распределение наиболее близко к распределению в соответствующей координате исходного изображения.
Агенты перемещаются до тех пор пока индекс структурного сходства (SSIM) изображений будет увеличиваться.

## Multiprocessing

Задача разбивается на процессы, каждый из которых выполняет указанные выше действия.
Раз в некоторое количество итераций распределения в точках матриц, полученных из процессов усредняются.
Процессам передается обновленная матрица.

## Анализ

Также представляется зависимость скорости работы при использовании разного числа процессов.
По результатам работы при использовании только 1 процесса сохраняется SSIM между полученной матрицей и исходным изображением.
При запуске двух и более процессов алгоритм останавливается при достижении данного значения.
